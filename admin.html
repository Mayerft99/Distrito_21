📋 Instrucciones de setup

Sigue estos pasos para configurar Supabase por primera vez:

  1. Ve a supabase.com y crea una cuenta gratuita.
  2. Crea un nuevo proyecto (elige la región más cercana).
  3. En tu proyecto ve a SQL Editor y pega el script del archivo supabase-setup.sql.
  4. Ve a Project Settings → API y copia la URL y la anon key.
  5. Pégalas en el formulario de la izquierda y haz clic en Guardar y conectar.
  6. ¡Listo! Ya puedes agregar productos.

📦 Storage para imágenes

Para subir imágenes de productos:

  1. En Supabase ve a Storage y crea un bucket llamado productos.
  2. Haz el bucket público (Public bucket).
  3. Las imágenes se subirán automáticamente al agregar productos.
